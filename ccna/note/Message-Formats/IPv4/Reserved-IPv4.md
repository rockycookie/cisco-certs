# Reserved IPv4 Addresses

## Private
- Class A
    - `10.0.0.0/8`
- Class B
    - 172.16.0.0/16
    - 172.17.0.0/16
    - 172.18.0.0/16
    - ...
    - 172.31.0.0/16
    - which is --> `172.16.0.0/12`
- Class C
    - 192.168.0.0/24
    - 192.168.1.0/24
    - 192.168.2.0/24
    - ...
    - 192.168.255.0/24
    - which is --> `192.168.0.0/16`

## Loopback
- 127.0.0.0/8
- 127.0.0.1 --> most commonly used for localhost
