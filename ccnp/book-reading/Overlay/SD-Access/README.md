# Software-Defined Access

## 4 layers
- Physical Layer
    - switches, routers, servers, wireless LAN controllers (WLCs), and wireless access points (APs)
- Network Layer
    - underlay and overlay networks
    - made available to the controller layer
    - its sole purpose is to transport data packets between network devices for the SD-Access fabric overlay
- Controller Layer
    - it provides all of the management subsystems for the management layer
- Management Layer
    - user interface/user experience (UI/UX) layer
